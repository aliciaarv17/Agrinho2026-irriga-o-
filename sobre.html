Objetivo Geral

Desenvolver uma plataforma web que auxilie na tomada de decisão sobre irrigação, incentivando o uso consciente da água, a redução de desperdícios e o equilíbrio entre produção agrícola e preservação ambiental em Icaraíma e região.

Problema

Em períodos de estiagem e calor intenso, muitos produtores precisam irrigar suas lavouras. Porém, a irrigação feita sem planejamento pode gerar desperdício de água e aumento dos custos.

Solução

O Gota Certa analisa dados simples da lavoura e gera uma recomendação de irrigação com base no Índice Gota Certa.

Impacto Ambiental

O sistema incentiva o uso racional da água e mostra a economia estimada em litros, ajudando a preservar recursos hídricos.

Relação com Icaraíma

O projeto foi pensado para a realidade agrícola de Icaraíma, no Noroeste do Paraná, região em que há produção rural, calor intenso em determinados períodos e necessidade de práticas sustentáveis para o uso da água.

Tecnologias Utilizadas

  • HTML para a estrutura das páginas.
  • CSS para o visual, responsividade e identidade do projeto.
  • JavaScript para interatividade, filtros e comunicação com o backend.
  • Node.js e Express para criação do backend.
  • Arquivo JSON como banco de dados simples para salvar as simulações.
  • Chart.js para geração dos gráficos do dashboard.

Originalidade

A originalidade do Gota Certa está em transformar dados simples do dia a dia agrícola em uma recomendação sustentável de irrigação. Além de calcular, o sistema educa, registra histórico, apresenta dashboard ambiental e valoriza a realidade local de Icaraíma.
